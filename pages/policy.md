# 독서화랑 운영 정책 가이드

**버전:** 2026.02 (Rev.2)
**대상:** 독서화랑 운영팀 및 CS 담당자

---

## 1. 온라인 서비스 및 계정 관리

### 1.1. 로그인 및 정보 수정
- **ID:** 기존 논술화랑 ID와 연동되며 변경 불가.
- **비밀번호:** 독서화랑 전용 비밀번호 설정 가능.
- **닉네임:** 7일 단위로 변경 가능 (중복 불가).
- **자녀 정보:** 이름, 성별, 학교명, 출생년도 수정 가능 (기존 화랑 ID 연동 정보는 변경 불가).

### 1.2. 아이디/비밀번호 찾기
- **아이디 찾기:** 가입 시 등록된 이름/휴대전화 번호로 인증 후 찾기 가능 (일부 마스킹 처리).
- **비밀번호 찾기:** 가입 시 정보로 인증 후 '재설정' 진행 (보안상 기존 비밀번호 안내 불가).

### 1.3. 자녀 선택 (다자녀 계정)
- 로그인 시 자녀 수에 따라 화면이 분기됨.
- 2명 이상일 경우 자녀 선택 화면이 노출되며, 헤더(GNB)를 통해 자녀 전환 가능.

### 1.4. 회원 탈퇴
- 마이페이지 > 정보 수정에서 가능.
- 독서화랑 사이트에서만 탈퇴되며, 기존 화랑 사이트 ID는 유지됨. (단, 독서화랑 재가입 불가)

## 2. 기본 개념 및 레벨 시스템

### 2.1. 레벨 구성 (총 30단계)
| 과정명 | 대상 | 레벨 범위 | 주요 활동 |
| :--- | :--- | :--- | :--- |
| **정독과정 (화랑도서관)** | 6세 ~ 초3 | Lv.1 ~ Lv.15 | 책을 깊이 있게 읽고 어휘/감정 활동 수행 |
| **다독과정 (탐험도서관)** | 초4 ~ 초6 | Lv.16 ~ Lv.30 | 다양한 분야의 책을 읽고 퀴즈/노트/평점 수행 |

### 2.2. 독서 성향 및 완독
- **정독과정:** '선호 분야' 부여 (256개 항목).
- **다독과정:** '탐험가 칭호' 부여 (128개 항목).
- **도서 완독:** 독서 활동을 완료하여 시스템상 '읽음' 처리되는 것. (레벨업의 기준)

### 2.3. 게시물 관리 정책
- **욕설/비속어:** 시스템상 자동 필터링 기능은 제공되지 않음.
- **제재:** 모니터링 또는 신고 접수 시 운영자가 수동 확인 후 삭제 조치. (부적절한 내용으로 획득한 재화는 회수될 수 있음)

## 3. 독서력 지수 진단 (레벨 테스트)

## 4. 정독과정 (화랑도서관) 상세 정책

### 4.1. 도서 이용 규칙
- **열람 범위:** 현재 레벨 ~ -2레벨 구간의 도서만 열람 가능.
- **검색:** 본인의 열람 가능 레벨 범위 내에서만 검색 가능.

### 4.2. 주요 독서 활동
1. **독후대화 (필수):**
   - 도서당 질문 카드 3개 제공.
   - 10자 이상 500자 이하 작성 (음성 입력 지원).
   - **1개 이상 작성 시 '도서 완독' 처리.**
2. **반복독서:**
   - 완독 후 스탬프 적립 가능 (최대 9회).
   - 9회 달성 시 '독서배지' 획득.
3. **정독어휘 (어휘통장):**
   - 도서별 주요 어휘 3개 제공.
   - 학생이 뜻과 예문을 직접 입력하여 '어휘통장'에 저장.
4. **감정발견:**
   - 도서를 읽고 느낀 감정 단어 선택 또는 직접 입력 (최대 3개).
5. **이미지 리터러시:**
   - 도서 삽화를 보고 학부모와 대화 유도 (별도 시스템 입력 기능 없음).

### 4.3. 정독퀴즈 및 승급 (시즌 정책)
- **응시 조건:**
    - **Lv.5 ~ Lv.10:** 21권 완독 시
    - **Lv.11 ~ Lv.15:** 15권 완독 시
    - *※ 적용 기간: 2025년 12월 ~ 2026년 2월*
- **방식:** 완독 도서 중 10권 랜덤 선정, 총 10문항 출제.
- **Pass 기준:** 8문항 이상 정답 시 승급 (Lv.15 Pass 시 다독과정 승급).
- **재시험:** 1일 3회 기회 제공 (익일 리셋).

### 4.4. 지성의 등불 (출석 보상)
- **승급:** 연속 3일 방문 시 단계 상승.
- **강등:** 연속 5일 미방문 시 단계 하락.

## 5. 다독과정 (탐험도서관) 상세 정책

### 5.1. 대륙 시스템 (레벨 구성)
| 대륙명 | 레벨 범위 | 비고 |
| :--- | :--- | :--- |
| 아프리카 | Lv.16 ~ Lv.17 | 시작 대륙 (진단 결과에 따라 다름) |
| 아시아 | Lv.18 ~ Lv.19 | |
| 유럽 | Lv.20 ~ Lv.21 | |
| 북아메리카 | Lv.22 ~ Lv.23 | |
| 남아메리카 | Lv.24 ~ Lv.25 | |
| 오세아니아 | Lv.26 ~ Lv.27 | |
| 남극 | Lv.28 ~ Lv.29 | |
| 미지의 대륙 | Lv.30 | 마지막 단계 |

### 5.2. 재화 시스템 (보상)
1. **지성의 별 (완독 기준 포인트):**
   - 획득처: 독서퀴즈(최대 3개) + 독서노트(최대 2개) + 도서평점(최대 1개).
   - **완독 처리 기준:** 한 도서에서 별 **총 5개**를 모두 획득해야 함.
2. **금화 (보상 화폐):**
   - 획득처: 지성의 별 10개당 1개 자동 환전, 레벨 미션, 부모님 응원.
   - 사용처: **선물 교환**, 독서퀴즈 재도전.

### 5.3. 주요 독서 활동 및 별 획득 기준
1. **독서퀴즈 (필수):**
   - 10문항 (어휘/사실/추론) 출제.
   - **Pass해야 다음 활동(노트, 평점) 가능.**
   - 재도전: 기본 2회 무료, 이후 금화 1개 소모.
2. **독서노트 (중요):**
   - **구성:** 도서당 2개의 질문 제공.
   - **지성의 별(2개) 획득 조건:**
     - 질문 2개 중 **최소 1개 이상을 150자 이상** 작성해야 함.
     - **주의:** 2개 질문을 각각 짧게 쓰거나, 150자 미만으로 작성 시 별 2개 획득 불가 (완독 처리 안 됨).
     - 별 1개씩 분할 지급되지 않음 (0개 또는 2개).
3. **도서평점:**
   - 평점(5점 척도) 및 이유(5~20자) 작성 시 별 1개 획득.

### 5.4. 보호자 설정 (학부모 기능)
- **선물 설정:** 금화 10/20/30개 교환 가능 선물 직접 등록 (쿠폰 발행).
- **응원 금화:** 자녀에게 메시지와 함께 금화 전송 (월 충전형).
- **도서 완독 기준 설정:** 자녀 수준에 맞춰 별 획득 난이도 조절 가능.

### 5.5. 레벨 승급 (무시험)
- **승급 기준:** 별도의 승급 퀴즈 없이 **레벨별 필수 완독 권수** 달성 시 즉시 승급.
- **필수 완독 권수:**
    - **Lv.16 ~ Lv.21:** 21권
    - **Lv.22 ~ Lv.23:** 18권
    - **Lv.24 ~ Lv.26:** 14권
    - **Lv.27 ~ Lv.29:** 10권
- **리포트:** 승급 시 기존 레벨 독서활동 보고서 자동 생성.

## 6. 과정 승급 및 리포트

### 6.1. 과정 승급 (정독 → 다독)
- **조건:** 정독과정 Lv.15 퀴즈 Pass 시 '탐험 티켓' 획득.
- **절차:** 2차 독서력 지수 진단(다독용) 진행 후 Lv.16~18 배정.
- **데이터:** 승급 후에도 '화랑도서관' 메뉴를 통해 이전 정독 활동 기록 조회 가능.

### 6.2. 독서활동 보고서
- **발행 시점:** 레벨 Pass 시 자동 생성.
- **정독 보고서:** 읽은 분야 분석, 반복 독서 달성률, 정독퀴즈 점수 등.
- **다독 보고서:** 도서 달성률(분야별), 퀴즈 유형별 분석, 중간 성향 분석 결과 등.
